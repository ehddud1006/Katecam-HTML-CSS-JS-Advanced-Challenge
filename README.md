## ✅ 참여 방법
1. 이 저장소를 `fork` 한다.
2. 생성된 원격 저장소에 `github ID`로 폴더를 생성한다.
3. 생성된 폴더에 클론코딩한 소스코드나 공부기록을 업로드 한다. (하루에 적어도 DAY 하나 분량)
4. 원본 저장소로 정해진 분량에 대한 `Pull Request`를 한다. (매주  **월,수,금 자정**에 Merge예정)
5. ***주의!! 기존 폴더를 지우면 안됩니다!!*** 개인 폴더 내부만 수정해주세요 :)
